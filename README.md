# 🐾 PetCare AI - Apresentação Profissional

Uma apresentação moderna e interativa do aplicativo PetCare AI.

## 🚀 Como executar

```bash
cd petcare-ai-apresentacao
npm start
```

## 📱 Navegação

- **Setas**: Navegar entre slides
- **Espaço**: Próximo slide
- **Escape**: Sair da tela cheia
- **F11**: Tela cheia
